# 🗒️ Notes App

A native Android notes application built from scratch with Kotlin, Jetpack Compose, and Room as a learning project on Android development fundamentals.

## 💻 Technologies

- `Kotlin`
- `Jetpack Compose`
- `Room Database`

## ✨ Features

- Add new notes with title and content
- View all notes in a staggered grid layout on the home screen
- Edit existing notes with auto-save on navigation
- Remove notes with confirmation dialog

## 💡 Process

This project was built as a hands-on learning experience using Kotlin, Jetpack Compose for UI, and Room Database for local data persistence. The app follows Android best practices with proper separation between UI components, data access objects (DAO), entities, and constants, implementing state management with `mutableStateOf` and `TextFieldState()` along with exception handling.

## 🛠️ Running the Project

1. Clone the repository and open it in Android Studio
2. Ensure you have the Android SDK installed (minimum SDK version as specified in build.gradle)
3. Sync the Gradle dependencies
4. Run the app on an emulator or physical device
